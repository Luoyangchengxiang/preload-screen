# preload-screen monorepo

> 极简单易用的 **React / Vue 首屏加载动画插件**  
> Zero-dependency、支持自动/手动模式、可自定义文案与动画样式。

## 🚀 快速开始

```bash
# 安装依赖
pnpm install

# 同时启动 React & Vue 示例
pnpm dev

# 构建所有包
pnpm build

# 跑测试
pnpm test

# 清理产物
pnpm clean

# 发布插件（自动构建+测试+npm publish）
pnpm release  
```
